# MyBlog
基于Struts+Hibernate+BootStrap+EasyUI的个人博客系统 包括前端和后台 编译器采用IDEA

数据库使用MySQL<br>
###前端包括：<br>
1.登录注册<br>
2.博主的个人信息<br>
3.博客浏览（主要部分，内容很多）<br>
4.个人技能<br>
5.博主联系<br>
###后台包括：<br>
1.登录<br>
2.个人中心（个人信息的查看、修改，登录密码修改）<br>
3.博客管理（写博客、查找博客（支持模糊查询）、查看博客（分页）、删除博客）<br>
4.评论管理（分页）、也可从博客管理页面进入评论区<br>

主要技术：<br>
前端主要用到BootStrap开源框架进行页面设计，此外还用到不少的JavaScript、CSS、JQuery、Struts2标签，<br>
完成一个页面良好、交互式良好的前端页面设计，前端的设计主要包括博主的个人信息以及博客的详细内容，<br>
这些内容博主在后台都可进行编辑和修改；<br>
后台设计主要用到Struts2、Hibernate、EasyU、JavaScript、JQuery、CSS完成对后台系统的设计，<br>
后台系统主要完成的模块包括站长统计、博主的个人信息管理、博客管理、评论管理等模块，后台总体设计也以优雅简洁的方式展现给用户。<br>

本系统的创新点包括：<br>
前后台整体设计的实现、<br>
BootStrap的使用、EasyUI的使用、<br>
数据库多表级联的设置、大量的多对一以及一对多的数据库设计、<br>
数据库注解的使用、站长统计的设置、<br>
博管理页面关于普通查看和模糊查询分页查看的设计、<br>
关于分页的PageBean的整体设计、博客管理页面多条信息同时删除的实现、<br>
进入评论区的设置、前端信息的滑动浏览设置、以及前端的弹出登录和注册，等等。<br>
